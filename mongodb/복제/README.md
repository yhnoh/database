1. replicaset 이름과 멤버 지정
- rs.initiate 를 사용하여 복제 셋 시작, 복제 셋 시작되며 나머지 구성을 전파하여 복제 셋이 형성된다.

```javascript
rs.initiate({
	 _id: "[복제이름]",
	 members: [
	   {_id: 0, host: "[컨테이너이름]"},
	   {_id: 1, host: "[컨테이너이름]"},
	   {_id: 2, host: "[컨테이너이름]"}
	 ]
});
```
- 복제를 구성하기 위해서는 _id, host가 필요하며 _id는 정수이며 복제셋 멤버간에 고유해야한다.
- 복제 셋은 프라이머리를 선출하며 rs.status()를 이용하여 복제 상태를 확인할 수 있다.
